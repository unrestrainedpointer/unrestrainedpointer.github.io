wordcloud 词图云

  1. wordcloud依赖于numpy包、pillow包和matplotlib包

    `python -m pip install numpy``

    `python -m pip install pillow``

    `python -m pip install matplotlib``

    `python -m pip install wordcloud``

  2. 三步使用

    创建对象：wc = wordcloud.WordCloud()

    加载文本：wc.generate()

    输出文件：wc.to_file('./1.png')

  3. 参数（只选了对我有用的研究）：

    width: default = 400

    height: default = 200

    max_words: default = 200

    stop_words: list of words that are excluded

    mask: shape of the wordcloud, need imread() function

    background_color: default = black

  4. 读取文本自动生成词云

numpy矩阵运算

  1. from numpy.matlib
  2. create a matrix:

    a = np.matrix([[1, 2]], [3, 4], [5, 6])

    转置：np.transpose(a)或a.transpose()或a.T

    迹：a.trace(), sum(a.diagonal())，其中a.diagonal()是获取主对角线元素

    秩：np.linalg.matrix_rank(a)

    行列式：np.linalg.det(a)

    逆矩阵：np.linalg.inv(a)

    相乘：np.matmul(a, b)
